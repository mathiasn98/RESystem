## About The Project
Tools for SMEs to support requirements engineering.
It's easy to use and has intuitive user interface.
Features:
1. Define business goals
2. Define current business process
3. Choose template
4. Define projects business process
5. Define functional and non-functional requirements
6. Requirements acceptance
7. Download requirements

### Built With
* [Laravel](https://laravel.com/)
* [Webpack](https://webpack.js.org/)
* [bpmn-js](https://bpmn.io/toolkit/bpmn-js/)

### Prerequisites
For more detailed guide, you can check in [Laravel Docs](https://laravel.com/docs/7.x/releases)
* Laravel 7.12.0
* Node.js
* npm
* Composer
* Valet/Homestead
* MySQL 5.6+


### Installation
1. Install dependencies
    ```sh
   composer install
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Create database
4. Edit .env
5. Migrate database
    ```sh
    php artisan migrate
    ```
6. Run laravel
    ```sh
    php artisan serve
    ```
7. Access in link provided in step 5
